How It Works

From signup to doctor visit in minutes

Create Account

Sign up with mobile OTP, email, or Google. Get your unique ZPT- Patient ID instantly.

2

Add medical history, allergies, medications, and emergency contact. Takes under 2 minutes.

Security & Trust

Built for real healthcare environments

Every architectural decision prioritises patient data security and compliance readiness.

🔑

Firebase Authentication

Industry-grade auth with mobile OTP, email/password, and Google sign-in.

🔒

Encrypted File Handling

Medical reports are encrypted in the browser using AES-GCM before upload. Architecture-ready for managed keys.

📋

Secure Patient Records

Firestore Security Rules ensure patients can only access their own data. No cross-patient data leaks.

🏗️

Role-Ready Architecture

Designed for patient / doctor / admin roles. Ready to extend with server-side Firebase token verification.
